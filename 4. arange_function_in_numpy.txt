APPROACH 2: arange() function
=============================
Syntax 1: varname = numpy.arange(value)
-> Gives range of values from 0 to value-1 in forward direction with default step size of 1

Syntax 2: varname = numpy.arange(start,stop)
-> Gives range of values from start to stop-1 in forward direction with default step size of 1

Syntax 3: varname = numpy.arange(start,stop,step)
-> Gives range of values from start to stop-1 by maintaining equal interval of step either in forward direction or backward direction with programmer step 

#step can be int or float

NOTE:
1. Here, arrange() always build 1D ndarray object only and unable to build multidimensional ND array (by default)
2. In order to get multidimensional ND array, we must reshape the ND array object by using 'shape' attribute or by using reshape() method

CONVERSION TO HIGHER DATATYPE AND MEMORY ALLOCATION
====================================================
Without using dtype keyword argument:
-------------------------------------
-> If we convert a list containing heterogeneous data to ndarray object, the datatypes of the elements will be converted to the next higher type

Ex1:
lst = [100, "RS", 34.56, True, 2+3j, False]
arr = numpy.array(lst)
print(arr) -> ['100' 'RS' '34.56' 'True' '(2+3j)' 'False']
print(type(arr)) -> <class 'numpy.ndarray'>
print(arr.dtype) -> <U64 

Explanation: print(arr.dtype) -> <U64 
-> This is because:
	'100' : 100 in numpy is int32 datatype => '100' is U32 datatype
	'RS': string representation in numpy is U => 'RS' is U2 datatype
	'34.56': 34.56 in numpy is float64 datatype => '34.56' is U64 datatype
	'True': True in numpy is bool datatype
	'(2+3j)':2+3j in numpy is complex128 **
	'False': False in numpy is bool datatype
Since, U64 is highest, print(arr.dtype) gives U64 as result

With using dtype=object as keyword argument:
--------------------------------------------
-> If we convert a list containing heterogeneous data to ndarray object (using dtype=object), the datatypes of the elements will be converted to supertype 'object'

Ex2:
lst = [100, "RS", 34.56, True, 2+3j, False]
arr = numpy.array(lst, dtype=object)
print(arr) ->  [100 'RS' 34.56 True (2+3j) False]
print(type(arr)) -> <class 'numpy.ndarray'>
print(arr.dtype) -> object **

NOTE:
1. None is used in python as a missing value (None belongs to <class 'NoneType'>
2. In numpy, equivalent to None is np.nan (np.nan belongs to <class 'float'>
